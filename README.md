## Development

React IDE :  
https://atom.io/  

Install node modules :  

    $ npm install  

Web pack your files :

    $ webpack

Start server :  

    $ node server.js  

Shut down current server in terminal :

    $ Ctrl + C

Automatic webpack bundling :

    $ webpack -w (it will listen to every changes mede in your files)
    $ node server.js (run in an other CLI windows)
