## Development

React IDE :  
https://atom.io/  

Install node modules :  

    $ npm install  

Start server :  

    $ node server.js  

Web pack your files :

    $ webpack ./public/app.js ./public/bundle.js

Shut down current server in terminal :

    $ Ctrl + C
